# Retail Sales Forecasting Dashboard

This project analyzes Walmart weekly retail sales using time series forecasting techniques.

## Models Used
- ARIMA
- SARIMA
- SARIMAX

Among these models, **SARIMA provided the best forecasting performance**.

## Features
- Time series analysis of Walmart sales data
- Model comparison using RMSE and MAE
- Interactive Streamlit dashboard
- Store selection for forecasting
- Future sales prediction

## Technologies Used
- Python
- Pandas
- Matplotlib
- Statsmodels
- Streamlit

## How to Run the App

Install dependencies:

pip install -r requirements.txt

Run the dashboard:

streamlit run app.py
